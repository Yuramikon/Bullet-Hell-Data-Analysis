### <span style="color: #2196F3;">Real-World Problem & Significance</span>
The **"Bullet Hell"** genre, which originated in Japan's arcade culture, has become increasingly difficult for players to discover because its Steam tag has been "hijacked" by unrelated game genres. Since **Steam** plays a vital role in the continued survival and accessibility of this genre today, distinguishing between games that remain true to their roots and those that do not is essential for both preservation and user experience.

## 2. Dataset Description

The foundation of this dataset is derived from the **Steam Game Recommendations** dataset hosted on Kaggle: 
[Kaggle Link](https://www.kaggle.com/datasets/antonkozyriev/game-recommendations-on-steam)

To align with the project goals, I performed a custom extraction by filtering all titles containing the **"Shoot'em up"** and **"Bullet Hell"** tags. Following the extraction, I manually labeled each entry based on their adherence to the genre's traditional roots.
